Aprende HTML

HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear y organizar el contenido de las páginas web. Usando etiquetas o marcadores, HTML define elementos como texto, imágenes, enlaces, botones y otros componentes visuales en una página web. En otras palabras, HTML es como la estructura o el esqueleto de una página web, diciéndole al navegador cómo mostrar el contenido.
